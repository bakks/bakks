## Keyboard Cheat Sheet

### tmux

#### Normal Mode

**C-b**
: Next pane

**C-x**
: Previous pane

**C-g**
: New pane split horizontally

**C-d g**
: New pane split vertically

**C-f**
: Next pane layout

**C-l**
: Kill current pane

**C-d h**
: Resize pane left

**C-d t**
: Resize pane down

**C-d n**
: Resize pane up

**C-d s**
: Resize pane right

**C-d d**
: Swap pane forward

**C-d D**
: Swap pane backward

**C-d r**
: Reload tmux configuration

**C-u**
: Enter copy mode

#### Copy mode

Allows you to manipulate the text of a tmux pane.

**H**
: Cursor left

**T**
: Cursor down

**N**
: Cursor up

**S**
: Cursor right

**t**
: Scroll down

**n**
: Scroll up

**C-t**
: Page down

**C-n**
: Page up

**V / C-v**
: Select text

**y / Y**
: Copy selected text
