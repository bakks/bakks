# [Dotfiles for Peter Bakkum](https://github.com/bakks/bakks)

## Keyboard Cheat Sheet

### tmux

#### Normal Mode

**C-b**
: Next pane

**C-x**
: Previous pane

**C-d g**
: New pane

**C-f**
: Next pane layout

**C-l**
: Kill current pane

**C-d h**
: Resize pane left

**C-d t**
: Resize pane down

**C-d n**
: Resize pane up

**C-d s**
: Resize pane right

**C-d d**
: Swap pane forward

**C-d D**
: Swap pane backward

**C-d r**
: Reload tmux configuration
